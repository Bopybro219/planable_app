Data rights

Your data choices

A simple guide to the requests Planira can review today, without pretending there is a bigger self-serve system than there really is.

Requests Planira can review

Access You can ask what account, search usage, API key records, contribution history, and related records Planira currently holds about you.
Correction You can ask Planira to correct inaccurate account or contribution information where it is practical to do so.
Deletion You can ask Planira to review deletion of account data. Some records, such as moderation history, API usage logs, or audit logs, may need to be kept for service integrity or abuse review.

What is not self-serve yet

Planira does not currently offer a polished self-serve export or deletion dashboard. Requests still need a manual review against the way the product works today.

How to ask

Use the contact details Planira publishes on the site when you need help with access, correction, or deletion. Today that includes {{ contact_phone }} in the footer. If a request touches moderation, abuse prevention, or financial records, Planira may need to review it carefully before making changes.
